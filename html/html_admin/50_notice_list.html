      기록
      • 즐겨찾기 탐색, 검색한 기록을 즐겨찾기 합니다. 다수의 즐겨찾기 목록을 생성할 수 있습니다.
      • 다운로드 탐색, 검색한 기록의 목록을 다운로드한 사용자 기록입니다. 개별 기록을 다운로드한 목록도 포함되어 있습니다.
      • 즐겨찾기 공개 사용자가 공개한 즐겨찾기는 베타기능입니다.
    • 요청관리
      요청목록
      • 요청목록 일반문의, 기록정보 오류신고, 개발오류 신고 영역의 요청이 이곳으로 수집됩니다.
    • 통계
      통계
      • 기본통계 Engine과 동기화한 데이터에 대한 기본 통계입니다.
      • 사용자 통계 로그인, 비로그인 사용자가 특정 영역의 활동을 했을 때 수집되는 사용자 통계의 전체 리스트입니다.
    • 포스트관리
      포스트관리
      • 공지사항 공지사항을 작성합니다.
      • 뉴스 외부에 전달하는 뉴스를 발행합니다.
      • 건축과정 오픈 전 건축과정을 소개합니다.
      • 칼럼 기록원 및 기록과 관련된 이슈를 다룹니다.
    • 회원관리
      회원관리
      • 이메일 수집목록 홈페이지 메인에서 수집되는 비회원의 이메일 목록입니다.
      • 회원목록 회원가입한 회원 목록입니다.
      • 회원유형분리 회원유형을 생성하거나, 분리해 서비스 개발에 필요한 데이터를 확보합니다.
    • 관리자관리
      관리자관리
      • 관리자목록 WEB 시스템의 관리자, 운영대응 담당자의 목록입니다.
    • F/A관리
      F/A관리
      • 리서치가이드 리서치 가이드는 자료 조사를 시작하기 위한 안내 지도입니다.
      • 사용자 가이드 특정한 사용자들에게 디지털아카이브를 안내하는 기본적인 도움말입니다.
